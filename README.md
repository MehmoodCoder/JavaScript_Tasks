# JavaScript Fundamentals - Core Demos

This repository contains 3 small interactive JavaScript projects built as part of my learning journey to master JavaScript fundamentals.

## 📅 Due Date
* **June 16, 2026**

---

## 🚀 Projects Included

1. **Counter Application**
   * A simple app to increment, decrement, and reset numeric values.
   * *Concepts used:* DOM Manipulation, Event Listeners.

2. **To-Do List**
   * An interactive list where users can add and remove daily tasks.
   * *Concepts used:* Array methods, Dynamic DOM creation, Form Handling.

3. **Quiz Application**
   * A dynamic quiz app that presents questions and checks answers.
   * *Concepts used:* Objects, Conditional Logic, State Management.

---

## 🧠 Key Learnings & Concepts Covered

Through these demos and learning resources, I explored the following core JavaScript mechanics:

* **Scope, Closures, & Hoisting:** Understanding how variables are accessed and memory allocation works in JS.
* **DOM Manipulation:** Selecting and updating HTML elements dynamically using JavaScript.
* **Event Handling:** Capturing user actions like clicks and form submissions.
* **Asynchronous JS (Promises):** Understanding how to handle asynchronous operations efficiently.
